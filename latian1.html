apa itu css

CSS atau Cascading Style Sheet adalah dokumen web yang memillki fungsi mengatur tampilan dokumen web lain, yastu HTML. Beberapa contoh pemanfaatan CSS pada HTML adalah untuk mengatur warna latar belakang, gaya huruf, jarak antar elemen, dan masih banyak lagi.
CSS terdiri atas kumpulan kode-kode yang keberadaannya tidak menggantikan HTML melainkan sebagai pelengkap agar tampilan dokumen HTML menjadi lebin indah dan dinamis. Berikut ini adalah contoh dokumen CSS.

bagaimana menggunakan css:

  1. Inline Styles: Menuliskan kode CSS langsung pada tag yang akan diubah atau diatur style-nya
  2. Internal/Embedded Styles: Menuliskan kode CSS pada bagian header dokumen HTML
  3. Eksternal/Linked Styles: Menuliskan code CSS pada file terpisah dan dipanggil melalui tag link dari dokumen HTML
